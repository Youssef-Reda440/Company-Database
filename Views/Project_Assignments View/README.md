# vw_ProjectAssignments View

## Description
This view shows the assignment of people to projects along with their roles.

## Purpose
- Used by management and project managers.
- Tracks who is working on each project.

## Data Included
- Project name
- Person full name
- Person type (Employee or Freelancer)
- Role in project
- Assignment start date

## Source Tables
- Project
- Project_Assignment
- Person
