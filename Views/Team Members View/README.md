# vw_TeamMembers View

## Description
This view shows all members assigned to each team, including employees and freelancers.

## Purpose
- Used by team leaders and management.
- Provides a unified view of team composition.

## Data Included
- Team name
- Department name
- Member full name
- Member type (Employee or Freelancer)

## Source Tables
- Team
- Department
- Employee
- Freelancer
- Team_Freelancer
- Person
