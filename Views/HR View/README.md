# vw_EmployeeProfile View

## Description
This view provides a complete profile for each employee in the company.
It combines personal information with job, team, and department details.

## Purpose
- Used by HR and management.
- Simplifies access to employee-related data without querying multiple tables.

## Data Included
- Employee full name and email
- Job title and salary
- Hire date
- Team name
- Department name

## Source Tables
- Person
- Employee
- Team
- Department
